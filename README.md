# Agentss Booking Confirmation Page

Standalone thank you page displayed after demo booking confirmation.

## Features

- Success animation with pulsing orange checkmark
- Booking confirmation message
- Timeline card with 3 steps:
  - Calendar invite sent (complete)
  - Pre-call briefing (pending)
  - Demo & gift card unlock (future)
- Clean, minimal design
- Optimized for Vercel deployment

## Deployment

This page is deployed as a standalone static site on Vercel.

## Local Development

Simply open `index.html` in a browser or run a local server:

```bash
python3 -m http.server 3000
```
